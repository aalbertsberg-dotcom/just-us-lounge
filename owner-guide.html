One-Time Google Sheets Setup
Do this once ‚Äî about 15 minutes

You'll create a free Google Spreadsheet that acts as your event manager. The website reads it automatically ‚Äî no coding required after the initial setup.

  1. Go to sheets.google.com and sign in with a Google account. We recommend creating a dedicated one like justuslounge.events@gmail.com so access stays with the bar, not tied to anyone's personal account.

  2. Click the big + to create a blank spreadsheet. Name it Just Us Lounge Events.

  3. In the very first row (Row 1), type these column headers ‚Äî one per cell starting at A1. You can copy and paste the whole line:

    name schedule time cover emoji color description extra_info performers facebook age dress_code active

    Tip: Copying that line and pasting directly into cell A1 will auto-split it across all the columns.

  4. Starting at Row 2, add your first event. See Section 2 for what each column means.

  5. Click Share (top right corner). Under "General access," set it to Anyone with the link ‚Üí Viewer. This lets the website read your sheet without needing a password.

  6. You're done with the spreadsheet for now. Jump to Section 5 to connect it to the website.

Adding, Editing & Hiding Events
Your day-to-day workflow
‚ûï Adding a New Event

  1. Open your Google Sheet at sheets.google.com.

  2. Scroll to the first empty row below your existing events.

  3. Fill in the columns from left to right. At minimum you need: name, schedule, time, description, and active set to yes.

  4. The sheet saves automatically. The website picks it up within a minute or two ‚Äî no other steps needed.

‚úèÔ∏è Editing an Existing Event

  1. Open the sheet and find the row for that event.

  2. Click any cell and type your changes. The sheet saves automatically.

  3. Done. The website updates on its own.

üôà Hiding an Event Without Deleting It

  1. Find the event's row in the sheet.

  2. In the active column, change yes to no.

  3. The event disappears from the website immediately. The row stays in the sheet so you can re-enable it any time by switching it back to yes.

üí° Linking to a Specific Facebook Event

When you create an event on Facebook, copy the URL from your browser's address bar and paste it into the facebook column for that event row. The "View on Facebook" button on the website will then take visitors directly to that event ‚Äî not just your main page.

üë• Multiple People Can Edit

In Google Sheets, click Share and add anyone's email as an Editor. Lynn, Tamara, and Lysa can all manage events from their phones or computers at the same time.

Connecting the Sheet to Your Website
One technical step ‚Äî done once by whoever sets up the site

Until this is done, the website shows its built-in default events. After this, it reads live from your Google Sheet.

  1. In Google Sheets, look at the URL in your browser. It looks like this:
    https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit
    Copy the long bold part between /d/ and /edit. That's your Sheet ID.

  2. Open the website's index.html file in any text editor (Notepad on Windows, TextEdit on Mac, or VS Code). Use Find (Ctrl+F / Cmd+F) to search for: YOUR_SHEET_ID_HERE

  3. Replace YOUR_SHEET_ID_HERE with your actual Sheet ID. Save the file.

  4. Upload the updated index.html to your web host to replace the old one. If you're not sure how, your web host's support team can walk you through it ‚Äî it's a simple file upload.

‚úÖ How to Confirm It's Working

Open the website and look at the Events section. If you see a green "Live ¬∑ Auto-synced from Google Sheets" badge below the heading, it's connected and working.

‚ö†Ô∏è Keep the Sheet Public

The Google Sheet must stay set to Anyone with the link ‚Üí Viewer. If that setting gets changed to private, the website can't read it and will fall back to showing the default built-in events until it's made public again.

Connecting the Mobile Form
One extra step so the form saves directly to your sheet
‚ÑπÔ∏è What This Does

The admin.html form lets anyone on the team add events from their phone without opening a spreadsheet. To actually save data into your Google Sheet, you need a free Google Apps Script "web app" that acts as the bridge between the form and the sheet. It's a one-time copy-paste setup.

  1. Open your Google Sheet. Click Extensions ‚Üí Apps Script in the top menu.

  2. Delete any code already in the editor. Copy and paste this entire script:

    function doPost(e) { try { var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Events"); var data = JSON.parse(e.postData.contents); var cols = ["name","schedule","time","cover","emoji","color", "description","extra_info","performers","facebook", "age","dress_code","active"]; var row = cols.map(function(k){ return data[k] || ""; }); sheet.appendRow(row); return ContentService .createTextOutput(JSON.stringify({result:"success"})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({result:"error",error:err.message})) .setMimeType(ContentService.MimeType.JSON); } } function doGet() { return ContentService .createTextOutput(JSON.stringify({result:"ok"})) .setMimeType(ContentService.MimeType.JSON); }

  3. Click Save (the floppy disk icon or Ctrl+S).

  4. Click Deploy ‚Üí New deployment.

  5. Click the gear icon next to "Type" and choose Web app.

  6. Set Execute as: Me ¬∑ Set Who has access: Anyone. Click Deploy.

  7. Google will ask you to authorize ‚Äî click Authorize access, choose your Google account, and click Allow. This is safe ‚Äî it's your own script accessing your own sheet.

  8. Copy the Web app URL that appears (looks like: https://script.google.com/macros/s/ABC.../exec).

  9. Open admin.html in a text editor. Find this line:
    const SCRIPT_URL = 'YOUR_APPS_SCRIPT_URL_HERE';
    Replace YOUR_APPS_SCRIPT_URL_HERE with your URL. Save and re-upload the file.

‚úÖ Test It

Open admin.html on your phone. Fill in a test event with "active" set to No, hit Save. Check your Google Sheet ‚Äî the new row should appear within a few seconds. Then delete that test row and you're good to go.

‚ö†Ô∏è If You Update the Script Later

If you ever need to edit the Apps Script code, go back to Extensions ‚Üí Apps Script, make your changes, then click Deploy ‚Üí Manage deployments ‚Üí Edit ‚Üí New version ‚Üí Deploy. The URL stays the same.

FAQ
Quick answers to common questions
How long until changes show up on the website?
Usually within a minute or two. If an edit isn't showing up, try a hard refresh: Ctrl+Shift+R on Windows or Cmd+Shift+R on Mac.
Can I have the same recurring event as multiple cards?
Yes ‚Äî just add it as separate rows. If Karaoke on Tuesday and Friday each need their own card with different details, add two rows.
What if the website stops showing my Sheet events?
Check two things: (1) Is the Sheet still shared as "Anyone with the link - Viewer"? (2) Is the Sheet ID in index.html still correct? The website will always fall back to the built-in default events if something goes wrong, so it never goes completely blank.
Is Google Sheets really free?
Yes ‚Äî completely free with any Google account, no limits, no expiration.
Can I change the About section, phone number, or other parts of the website?
Yes, but it requires editing the index.html file directly in a text editor. Search for the text you want to change, update it, save, and re-upload. It's straightforward ‚Äî just find and replace the text you see.
Who can I call if something breaks?
Reach out to whoever set up the site. Keep this guide handy ‚Äî most issues can be resolved by checking the two things listed in the "What if it stops working" answer above.
